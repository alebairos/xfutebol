# Architecture Decision Records

Lightweight ADRs for cross-repo decisions. Use this format:

```
docs/adr/NNN-short-title.md
```

Each ADR covers: **Context → Decision → Consequences**.

Only document decisions that are non-obvious or that future agents/devs would otherwise re-litigate.
